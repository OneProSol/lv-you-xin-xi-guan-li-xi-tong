### 一、相关技术
- 前端：`thymeleaf`、`HTML` / `CSS` / `Javascript`。
- 后端：`Springboot`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`jdk1.8`。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示
系统分为：用户、管理员。

用户功能：1. 登录、2. 注册、3. 首页、4. 旅游路线、5. 旅游景点、6. 餐饮住宿、7. 旅游车票、8. 旅游保险、9. 旅游攻略、10. 注意事项、11. 会员中心

管理员功能：1. 登录、2. 后台首页、3. 用户管理、4. 旅游路线管理、5. 景点管理、6. 酒店管理、7. 留言管理、8. 攻略管理、9. 车票管理、10. 保险管理、11. 订单管理、12. 数据分析
